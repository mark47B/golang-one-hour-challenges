# Simple-crawler
Напишите на Go программу-краулер, которая параллельно загружает содержимое нескольких веб-страниц, выводит их тела в консоль и корректно завершает работу по сигналу ОС (SIGINT/SIGTERM).

## Требования
- Параллельная загрузка
- Ретраи с экспоненциальным backoff
- Таймауты
- Graceful shutdown
- Обработка контекста
- Чтение тела ответа

go run .
