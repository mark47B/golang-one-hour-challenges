# fan-out / демультиплексор каналов / распределение значений
fan-out — это конкурентный паттерн, который читает значения из одного входного канала и распределяет их по нескольким выходным каналам.

Каждое значение, полученное из входного канала:
- должно быть отправлено во все выходные каналы (broadcast),
или
- должно быть отправлено ровно в один выходной канал (load balancing)

Тип распределения необходимо выбрать и явно зафиксировать в реализации (и описать в комментариях).
